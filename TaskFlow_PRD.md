# 产品需求文档 (PRD)

## 一、项目概述

### 1.1 项目名称
**TaskFlow** - 极简任务管理与番茄钟结合的Web应用

### 1.2 项目定位
一款面向个人用户的时间管理与任务规划工具，帮助用户高效管理日常任务并培养专注习惯。

### 1.3 核心价值
- 简洁直观的界面设计，降低学习成本
- 任务管理与番茄工作法深度结合
- 数据本地存储，保护用户隐私

---

## 二、功能需求

### 2.1 任务管理模块

#### 2.1.1 任务创建
- 用户可创建新任务，设置任务标题
- 任务支持设置优先级：高/中/低三级
- 任务支持设置截止日期（可选）
- 支持任务分类标签（最多3个标签）

#### 2.1.2 任务列表
- 展示所有未完成任务
- 支持按优先级、截止日期、创建时间排序
- 显示任务剩余数量统计

#### 2.1.3 任务操作
- 标记任务为已完成
- 编辑任务内容
- 删除任务
- 一键清空已完成任务

### 2.2 番茄钟模块

#### 2.2.1 番茄计时器
- 默认25分钟工作时段，5分钟休息时段
- 可自定义时长设置
- 计时器暂停/继续功能
- 计时结束音效提醒

#### 2.2.2 番茄记录
- 记录每个番茄钟对应的任务
- 显示今日完成的番茄数统计

### 2.3 数据统计模块

#### 2.3.1 统计面板
- 今日完成任务数
- 今日完成番茄数
- 本周任务完成趋势（柱状图）

## 三、技术选型

### 3.1 前端技术栈
- **框架**：Vue 3 (Composition API)
- **构建工具**：Vite
- **状态管理**：Pinia
- **样式方案**：CSS Variables + Scoped CSS
- **图表库**：ECharts（用于统计图表）

### 3.2 数据存储
- **存储方案**：localStorage
- **数据结构**：JSON格式

### 3.3 项目结构
```
src/
├── assets/          # 静态资源
├── components/      # 公共组件
│   ├── TaskItem.vue
│   ├── PomodoroTimer.vue
│   └── StatsChart.vue
├── stores/          # Pinia状态管理
│   └── taskStore.js
├── views/           # 页面组件
│   └── HomeView.vue
├── App.vue
└── main.js
```

## 四、界面设计

### 4.1 整体风格
- 极简主义设计
- 浅色主题为主
- 圆角卡片式布局

### 4.2 配色方案
- 主色：#4A90D9（蓝色）
- 背景：#F5F7FA（浅灰）
- 卡片：#FFFFFF（白色）
- 文字：#333333（深灰）
- 强调：#FF6B6B（红色，用于删除/高优先级）

### 4.3 布局结构
- 顶部：应用标题 + 导航
- 左侧：任务列表区域
- 右侧：番茄钟 + 统计面板
